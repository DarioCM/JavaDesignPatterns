--
El patrón decorador es un patrón de diseño estructural que permite añadir comportamientos a objetos individuales,
ya sea de forma estática o dinámica, sin afectar al comportamiento de otros objetos de la misma clase.
En esencia, permite adjuntar nuevos comportamientos o responsabilidades a objetos como decoradores.

--
The Decorator Pattern is a structural design pattern that allows behavior to be added to individual objects,
either statically or dynamically, without affecting the behavior of other objects from the same class.
In essence, it allows you to attach new behaviors or responsibilities to objects as decorators.
